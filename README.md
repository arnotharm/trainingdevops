## MarkDown - специальный язык для конспектов
> **Заголовки по размерам и метод вызова**
> ==
# Заголовок 1
## Заголовок 2
### Заголовок 3
И так далее. Заголовк 1 и 2 можно описать другими символами. * == (два равно подряд - Заголовок 1 | Заголовок 2 можно вызвать от двух черточек (--)
>
Заголовок 1
==
Заголовок 2
--

# **Жирность** и *Курсив*
Жирное слово - __Жирное слово__ | **Жирное слово**   \__Жирное слово__ | \**Жирное слово**
Курсив - _Курсив_ | *Крусив* \_Курсив_ / \*Крусив*

# **Метки / Цитаты / Код**
`Цитата, выделаеная`
Чтобы добавить код с визуальными эффектами: нужно поставить 3 обратные скожки "`" в на чале и в конце блока 
```
.head{
size: 12px;
}
```

# Контроль версий или GIT / GitHub

#### Этапы старта проекта:
-	Создать репозиторий в Git Hub
- В директории проекта создаем локальный репозиторий - **git init** (создается ".git")
- Добавляем нужные файлы или все в INDEX - **git add .** или **git add index.html**
>![](https://bramus.github.io/ws2-sws-course-materials/assets/xx/git-remote.png)

- Создание первого коммита - **git commit -m "Первый коммит"**
- Подключение к удаленному репозиторию - **git remote add origin https://github.com/**
